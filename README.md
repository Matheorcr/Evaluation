# Evaluation Dash 2025-2026

Application Dash realisee dans le cadre de l'evaluation.

## Prerequis

- Python 3.12+
- uv

## Installation

```bash
uv sync
```

## Lancer l'application

```bash
uv run app.py
```

L'application est ensuite disponible sur `http://127.0.0.1:8050`.

## Structure du projet

- `app.py` : point d'entree principal
- `pages/` : fichiers des pages (layout et callbacks)
- `assets/` : ressources statiques (CSS, images)
- `datas/avocado.csv` : jeu de donnees principal
- `expli1.md`, `expli2.md`, `expli3.md` : contenus Markdown fournis
